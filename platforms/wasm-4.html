WASM-4

Released: 2021

63 games

WASM-4 is a fantasy game console designed for creating small, retro-style games using WebAssembly. Launched in 2021, it provides a minimal virtual hardware platform featuring a 160x160 resolution display, 4-button input, and simple audio and sprite capabilities. Developers can write games in a variety of languages—including C, Rust, AssemblyScript, and more—then compile them to WebAssembly for cross-platform compatibility. WASM-4 games run directly in modern web browsers or via native runtimes, making it a popular choice for game jams, prototyping, and experimenting with low-level game development.
